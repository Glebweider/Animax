# Guide
## Installation FrontEnd

```bash
# transition to FrontEnd
$ cd '.\FrontEnd\'

# install
$ npm install
```

## Running the app

```bash

# start  
$ npx expo start

# start android
$ npm expo android

# start ios
$ npm expo ios

# start web
$ npm expo web
```
